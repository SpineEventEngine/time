public interface TemporalMessage<T extends TemporalMessage<T>> implements Temporal<T>

A Temporal implemented with a message.

Messages marked with the Temporal interface should use this type instead of using the Temporal directly.

To create a temporal message type:

  1. create a new interface derived from this one;
  2. specify the target message type as the type parameter;
  3. implement leftover abstract methods inherited from Temporal;
  4. mark the target message with the (is) option.
